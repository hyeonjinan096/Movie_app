# Movie_app

## 📚**프로젝트 개요**


[부스트코스]안드로이드 앱 프로그래밍

- 부스트코스에서 제공하는 서버로부터 영화 정보를 받아 보여주는 안드로이드 앱 입니다.
- 단계별 프로젝트를 수행했습니다.
<br>
<aside>
💡
1. 영화상세 화면 만들기
  
2. 좋아요와 한줄평 리스트

3. 한줄평 화면으로 전환하기

4. 영화목록과 바로가기 메뉴

5. 서버에서 영화정보 가져오기

6. 영화정보를 단말에 저장하기

7. 사진보기와 동영상 재생

8. 영화 앱을 더 즐겁게
<br>
</aside>
